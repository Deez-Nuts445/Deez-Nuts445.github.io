card.css is a CSS script that could help with creating sleek cards.


To use card.css, put in the script in your HTML code first

<link rel="stylesheet" href="https://rawcdn.githack.com/Deez-Nuts445/card.css/fb48ec3aadc780c613ce383c80578e1ebe6d8f10/card.css">

Then to use the card, create a div element with the class "card"

<div class="card"></div>

Then put a h3(title) and p(description) element

    <div class="card">
    <h3>Card Title</h3>
    <p>Card Description</p>
    </div>
